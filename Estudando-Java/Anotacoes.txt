recursividade é quando um metodo invoca uma nova execução de si mesmo após executado. 

- Fatoria com While: 
Trabalhando com a variavel long, o valor maximo que vc ira conseguir fatorar é 9223372036854775808 ou o fatorial de 37.
Por quê usar long? Neste caso, usamos long porque numeros fatoriais são grandes e em muitos casos é preciso usar bastante
espaço na memória para guarda-los.  

Neste caso no inicio do while, onde utilizamos a variavel que recebeu o numero para ser fatora do while(num>1) logo o a 
variavel num é recursiva pois até ela ser menor ou igual a 1 ela ira invocar uma nova execução de si mesma. 

- Fatorial com Recursividade:
 
É possivel cria classes com efeitos de laços de repetição sem declarar while, do while ou for, utilizando a chamada do méto
do (só que passando valores diferentes a cada passo).


- Classes e Objetos 
 
Classe é o como chamamos no mundo fora do computador como "tipos", exemplo se eu falar "Um felino perigoso esta a solta"
vc logo imagina um leão, um trigre... isso porque voce oq é um tipo de felino. 
Logo classes são tipos para os computadores. Por isso dizemos que Java é uma linguagem tipada. 

Deixando mais limpo uma classe pode ser utilizada para criar uma variavel e utilizar eu seu programa e para instanciar uma 
classe vc precisa do comando construtor "new", ex: Atleta jogador= new Atleta();

 .Intanciação 

 Usando uma construção de uma casa como exemplo, temos então que a planta é a  classe(tipo) dessa casa, e objetos dessa classe
seriam casas construidas com base nessa planta e instanciar uma classe seria o ato de criar uma casa de acordo as instruções 
dessa planta. 
 Ou seja, instaciar uma classe significa criar um objeto de acordo com as instruções dessa classe.
